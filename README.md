# Airline Customer Booking Analytics
An interactive Excel dashboard project that analyzes airline booking behavior, trip timing, and customer travel patterns across 64 destinations.
## Project Summary
This project explores airline customer booking data to uncover insights related to:
- Booking behavior
- Trip timing and duration
- Flight day and hour trends
- Lead time segments
- Destination-level activity
The dashboard was built in Excel and designed to present key metrics in a clear, business-focused format.
## Key Metrics
- **Average length of stay:** 23.04 hours
- **Peak flight day:** Monday
- **Peak flight hour:** 8
- **Average flight duration:** 7.28 hours
- **Destinations analyzed:** 64
## Insights
- Flight activity is strongest on Mondays.
- Booking behavior changes depending on lead time.
- Customer travel patterns vary across different trip types and sales channels.
- Excel can be used to build strong, decision-ready dashboards for business analysis.
